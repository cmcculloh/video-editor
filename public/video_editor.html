To use this:

        
    
            
  1. Edit this file and 
    video_editor.js
     and 
    ../edit_video.cjs
     to point at the desired video file (can be m4v, mp4, and maybe mkv)
    2. 
            
  2. Start the server by 
    npm videoeditor
    3. 
            
  3. Use the buttons on screen (forward/backward keys skip by 10 seconds, whereas the buttons skip by frame)
    4. 
            
  4. Copy the resulting timestamps into 
    tv-edit-list.cjs
    5. 
            
  5. Run 
    npm run doedit
    6.
